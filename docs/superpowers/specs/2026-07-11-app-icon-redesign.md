# ImageView 应用图标重设计

日期：2026-07-11  
状态：已确认

## 目标

替换当前细节过密、偏游戏插画风格的应用图标，建立更符合原生 macOS 图片浏览器定位的专业视觉形象。新图标需要在 Dock、Finder 和应用切换器的小尺寸场景中保持清晰辨识度。

## 视觉方向

采用“玻璃取景框”方向：深色烟熏玻璃圆角底座，中心使用简化的照片画框与山形光影符号。整体以蓝紫和青色边缘高光塑造层次，并加入克制的金属与玻璃材质表现。

## 构图与风格

- 使用 macOS 应用图标常见的圆角方形轮廓和居中构图。
- 主符号占据足够面积，轮廓简洁，缩小后仍能识别。
- 光照柔和、层次精致，不使用夸张霓虹或密集装饰。
- 不使用文字、水印、相机机身或写实镜头。
- 不采用复杂风景插画，避免与游戏和摄影编辑软件混淆。

## 交付规格

- 生成一张 1024 × 1024 像素的高质量主图。
- 主图保存为 PNG，并保留适合 macOS 图标显示的边缘与留白。
- 从主图生成工程使用的 `ImageView.icns`。
- 替换 `Sources/ImageViewApp/Resources/AppIcon-master.png` 与 `Sources/ImageViewApp/Resources/ImageView.icns`。
- 构建应用后检查图标资源已正确打包。

## 验收标准

1. 图标明确传达图片查看或画面浏览的用途。
2. 视觉质感比现有图标更克制、现代、专业。
3. 16–64 像素缩略显示时，主体轮廓仍然清楚。
4. 不含文字、水印、复杂背景或无关元素。
5. PNG 与 ICNS 均能被当前构建流程正常使用。
