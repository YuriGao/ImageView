# ImageView 画布平移边界设计

日期：2026-07-10
状态：已确认，待实现

## 规则

1. 图片放大后，平移偏移限制在仍让图片覆盖画布的范围内；任一轴图片小于画布时，该轴偏移固定为零。
2. 鼠标拖拽、滚轮平移和缩放后的锚点偏移都使用同一边界裁剪函数，避免跳变与图片离开视野。
3. 横向触控板滚动在适应窗口时直接切图；放大时仅当图片已到对应水平边缘且继续向外滑动才切图。
4. 到边缘的判断容差为 1 pt，避免浮点误差造成意外切图。

## 实现与测试

`ImageCanvasView` 公开纯几何的 `clampedOffset` 和 `isAtHorizontalEdge` 辅助方法。现有滚轮逻辑先平移并检测到边缘，再决定是否导航。测试覆盖大图边界、窄图居中、缩放后限制及边缘方向切图。
